Mengenai Semantic Elements di HTML 5

  • Article

    • Ketika ingin membuat layout untuk menampilkan blok artikel-artikel yang ada di website kita, tag article bisa kita gunakan dengan element header dan footer. karena tag article memiliki type block, artinya element ini memiliki lebar sesuai lebar browser jika kita tidak menentukan lebarnya.

  • Aside

    • Element ini bisa kita gunakan untuk melengkap layout utama, misal kita memiliki halaman blog maka kita bisa menambahkan aside dibagian bawah postingan untuk menjelaskan identitas penulis artikel. karena aside memiliki type/sifat block maka lebarnya sesuai page jika kita tidak menentukan lebarnya.
